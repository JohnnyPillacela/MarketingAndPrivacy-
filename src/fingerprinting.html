Browser Fingerprinting

Published by: Johnny Pillacela

Everyone has fingerprints. It is what identifies you from other people because it is unique to you. You know what also has fingerprints? Browsers! Well browsers don’t physically have fingers to have a fingerprint but they collect your personal information and make a fingerprint out of you. As most people know, websites that you visit online like Amazon, Tik Tok, and Instagram collect your personal information to run ad campaigns based of off what you like. The type of personal information that these websites collect from you can include:

  • What browser you’re using
  • Where and what time is it
  • How old you are
  • How long you spent looking at certain things on the website

But most of these things that they collect from you are very common things. Like you and your friend can have the same age, live in the same area, browse the internet using the Google Chrome browser, and spend looking at things almost the same amount of time. So how are marketers and advertises able to create a fingerprint on you. After all, fingerprints are unique to everyone and since you and your friend are very similar, that might be hard to distinguish online.

One of the tools that marketers and advertises use to collect your personal information is called JavaScript. JavaScript is a programming language that runs code in the background of a website without the user ever knowing about it. With JavaScript, websites can find the following information

  • Resolution of the screen
  • Orientation of the screen
  • Device type
  • Make and model of your device
  • Time zone
  • What plugins you have installed
  • Are you currently interacting with the site?
  • If you have Microsoft office installed in the browser
  • Can perform the font test (which identifies how your computer draws a specific letter)

Now, it is a combination of all these things that uniquely identify you. As you can see JavaScript is a big supplier of all of this information which means if you disable JavaScript in your browser then the website won’t be able to retrieve this information from you. However, disabling JavaScript in a browser can severely alter how a website looks like and possibly break the website. Overall, by identifying your age alone, a website doesn’t have any valuable information from you. However, with all this information collectively websites, marketers, and advertisers can create a unique fingerprint on you and save it so the next time you visit the same site they already know who you are.
